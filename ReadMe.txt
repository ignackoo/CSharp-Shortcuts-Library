C# Shortcuts Library

To Create and Read .url and .lnk shortucts in windows

*******************************************************************************

In project you must ...
Add a reference to “C:\Windows\system32\wshom.ocx”. (IWshRuntimeLibrary.dll)

Right click on References in Visual Studio Solution Explorer
Choose Add Reference... 
Navigate to the COM tab and find “Windows Script Host Object Model” 
Add it to project References (checkboxed)
Click OK
